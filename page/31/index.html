主定理

发表于 2021-11-15 分类于 Other
本文字数： 278 阅读时长 ≈ 1 分钟
标签：     数学

$$
If \quad T(n) = aT(\left \lceil \frac{n}{b} \right \rceil ) + O(n ^ d),
$$

$$
then:T(n) = \begin{cases}O(n ^ d) \quad if \quad d > \log_{b}{a}
\\O(n ^ d \log n) \quad if \quad d = \log_{b}{a}
\\O(n ^ {\log_{b}{a}}) \quad if \quad d < \log_{b}{a}
\end{cases}
$$
