Algorithm IV ST表

参考资料：mydcwfy

遍历到一个点的时候加在序列末尾，记当前位置为 $id(x)$ ，然后它的一个子树走到另一个子树时，也要将 $x$ 加在序列末尾。

得到的序列可以证明不会超过 $2 n-1$ 。

从 $x$ 到 $y$ 需要从 $lca$ 的一个子树走到另一个子树，所以 $id(x)$ 到 $id(y)$ 之间一定有 $lca$ 。

那么中间深度最小的点就是 $lca$ ，可以用 ST表 维护。

复杂度： $O(n \log n + T)$ 。洛谷模板 $2.65s$ 。
